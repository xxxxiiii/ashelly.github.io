vue-基础部分

发表于 2020-03-19 更新于 2020-03-21 分类于 学习笔记

基础部分

Vue是一个渐进式框架

渐进式意味着可以将Vue作为应用的一部分嵌入其中，带来更丰富的交互体验。

Vue的特点和Web开发中常见的高级功能：

  • 解耦数据和视图
  • 可复用组件
  • 前端路由技术
  • 状态管理
  • 虚拟DOM
v-on的修饰符
  • .stop 停止冒泡 => 调用event.stopPropagation()
  • .prevent 阻止默认事件 => 调用event.preventDefault()
  • :keyup 监听某个键盘的键帽
  • .once 只触发一次回调
计算属性和methods的不同

vue内部会缓存计算属性，计算属性没有改变值时多次调用只执行一次代码，而methods不会，调用几次就会执行几次代码

v-if和v-show的不同
  • v-if 当条件为false时，包含v-if指令的元素不会存在在dom中，v-show 当条件为false时，会给元素加一个行内样式：display:none。
  • 当需要在显示与隐藏之间切换很频繁时用v-show,切换频率不高时，用v-if
v-model
  • v-model背后本质是包含两个操作,v-bind绑定一个属性,v-on指令给当前元素绑定Input事件*
  • v-model的修饰符
    • .lazy 失去焦点或敲回车后才会更新
    • .number 将数据类型设置为number
    • .trim 去除空格修饰符
较少用的指令
  • v-once:首次使用时展示数据，数据改变时页面不会随着数据的改变而改变
  • v-pre：把文本原封不动的展示出来，不进行解析
  • v-cloak:vue解析前div中有v-cloak属性,解析后没有
